Да, вы можете использовать метод `<attribute>_changed?` для любого атрибута в модели, чтобы проверить, был ли этот атрибут изменен и сохранен. Включая имя пользователя.

Допустим, у вас есть модель `User` с атрибутами `:username` и `:email`. Вы хотите проверить, было ли изменено имя пользователя (`:username`). Вы можете сделать это следующим образом:

```ruby
user = User.find(1)  # Получаем пользователя из базы данных

# Проверяем, изменилось ли имя пользователя и было ли оно успешно сохранено
if user.username_changed?
  # Выполняем действия, если имя пользователя изменилось и было сохранено
  puts "Имя пользователя было изменено"
else
  # Выполняем действия, если имя пользователя не изменилось или не было сохранено
  puts "Имя пользователя осталось без изменений"
end
```

Таким образом, метод `<attribute>_changed?` может быть использован для отслеживания изменений любого атрибута в модели и проверки, были ли эти изменения сохранены в базе данных.
